**************************************************
*     Instructions for installation              *
**************************************************

	1.	Unzip the .ZIP archive

	2.	Open the file index.html in a text editor
		(Notepad or similar)

	3.	Edit the following information:
		- Website title inside the <title> tag
		- Website title inside the <div id="header">
		tag
		- Page content and navigation, marked with
		comments: <!-- CONTENT START --> to <!--
		CONTENT END --> and <!-- NAVIGATION START
		--> to <!-- NAVIGATION END -->
		- Your contact information inside the
		<div id="footer"> tag

	4.	Save the file under corresponding names
		(index.html, about.html, links.html etc.)

	5.	Upload all files and folder /images to your
		server.

	For more detailed information on installing the
	template, refer to my tutorial at the address:
	http://inobscuro.com/tutorials/read/31/
