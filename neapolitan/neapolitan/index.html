A New Template

from DemusDesign

Neapolitan is a new fun template from DemusDesign. It took a lot of self-control to create it without constantly heading to the freezer for some ice cream. New colors for me and I successfully made a template without any gradients (yay). It validates as XHTML 1.0 Strict. If you scroll down you will see tags for blockquotes, images and forms.

Feel free to use the template, but please include the link back to my site in the footer. Thanks, enjoy.
