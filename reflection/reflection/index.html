template info Another fluid, XHTML/CSS template for OWD. I decided to give a go at creating one that could be easily modified and this is what I came up with. The header/footer images are easy to change, along with the accent colour - just do a search in the style sheet for hex value #FC0. The template was tested in the usual, and looks good in all:

  • Firefox 1.0.3
  • Internet Explorer 6.0
  • Netscape 7.02
  • Opera 8.0

Images have full support without the need for tables. To align them left or right, use the .imgLeft or .imgRight classes. The headings will act as boundaries, so you don't have to worry about overflowing pics. It should also be fairly straightforward to use tables for layout if you find the need for them.

fluid css Most of the dimensions of the template are defined relative to the font size or using percentages, so increases/decreases in font size won't break the layout. The template also gracefully contracts/expands to different screen resolutions, so there are no gaps.

This is a free use template - hope it makes your life easier. If you do end up finding a use for it, send me an e-mail since it's interesting to see my templates in action.
