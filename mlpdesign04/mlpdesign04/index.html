Reminders when editing

Posted on 11 April 2007, 2:16 PM by webmasterneo | 0 Comment(s).

To keep the design from messing up, change items in the index.html file only when they are outside the <tags>. If you wish to edit the color, font, size, etc, do so within the stylesheet (default.css) file. Otherwise, the code may not validate or may not show up properly in other browsers.
