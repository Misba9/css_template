The changes in this alternative version, compared to the default version are:
  • Font size not set
  • Font-family not set

The reason for this is:

I am trying to get the focus for my templates on usability/accessability. The font size/family is not set so that the user can easily adjust their browser to how they like it. For more detail, please read the links below:
Why not to over-ride the reader’s font size

So while this can make the design less attractive, it is easier for those that want better control of their font. If you find the text too big, change it, or read the link below to learn how
Adjusting your text size

If anyone can give me links etc on how to improve accesability, I would be very happy to read them. Please use the email option on my OWD user page.
