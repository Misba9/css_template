About This Design

Well, this design was made because I wanted to make something that didn't involve hacks for the primary layout.

The left and right columns are set at 200px, and the center area is fully flexible to whatever size is needed. All fonts will increase appropriately because all are set relative using the base at the 76% universal sizing.

One image is used through this layout, and is simply repeated in different spots to show possible placement of graphics.

You can have multiple subsections in your left navigation, but the only ones that will show expanded are the subsections for the <ul> currently marked #active. The subsections for other sections of the menu will remain hidden.

This website is XHTML 1.0 Strict and has been tested in Firefox, IE 6 and IE 7 Beta 2, Opera, Konqueror, and Safari.
